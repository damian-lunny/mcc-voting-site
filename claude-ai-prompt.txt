Please create a website for Maguiresbridge Camera Club.

It should run on PHP and MySQL. The table definitions are below. The pages should use Bootstrap and Bootstrap Icons where appropriate.

The Vote page is where members can place their votes for up to five photographs. The first time in a session a member goes to the page, it should present a login box in order to enter a PIN. Without entering this correctly the rest of the pages will not be accessible. The PIN can be stored in the table "settings".

On entering the PIN successfully, every page should now have the following links:
  Vote
  Live Tally
  Archive
  Admin
  Logout

At the bottom of every page - including the PIN page, there should be a button for changing the page display to High Contrast. Once clicked it should instead say "Normal Contrast" and affect every page shown until clicked again.

Similarly, a button labelled "Larger Text" should increase the font size of the page text by 20%. Once clicked, it should change to "Smaller Text" and affect all pages until clicked again. Clicking "Smaller Text" should return the text size to the original.

Page: Vote

On entering the PIN successfully, the camera club member will be presented with a page allowing them to vote in three categories:
  Beginner
  Intermediate
  Advanced

Each section will only be available when enabled from the Admin page. when a category is closed, the voting page will display a message "Voting in this category is currently closed".

The voting sections will show boxes to enter a number. That number will correspond to a photograph being displayed in the club on competition night. Each member can choose up to five photographs to give points to.
Points will be allocated as follows:
  First place: 5 points
  Second place: 4 points
  Third place: 3 points
  Fourth place: 2 points
  Fifth place: 1 points

When the member clicks Submit, the results will be saved in the Votes table.

Page: Live Tally
Viewing this page will show the current results for the current active competition. It should show the Category and then the results for each photograph that has received a vote. The results should be counted up using the points allocated to show which photographs have the most points. The results should be displayed in a horizontal bar in descending order with the photograph with the most points at the top.

This page should auto-refresh every 10 seconds. The timestamp for the most recent refresh should be shown in the format "hh:mm:ss".

At the bottom of this page should be an option visible only to Admin: "Clear all votes". Clicking this should clear all votes - for each enabled category.

Page: Archive
This page will show all the currently archived results. Selecting an archive will show the results for each category in the competition. It will show the photo number and the number of points it received. Additionally, each photograph will have two boxes: Photographer and Photo Title. Each photo will have a Save button which when clicked will save the Photographer and Photo Title against the corresponding photo number - updating the Archived Results table in the process.

Below each category in the displayed archive, three fields will be displayed: two "Highly Commended" fields and one "Commended" field. This will save the photo number from the competition as well as the Photographer Name and Photo Title - saved in to the corresponding competition in the Archive.


Page: Admin
This page will have multiple sections:
  Statistics
  Site Access
  Category Control
  Photo Settings
  Competition Management
  Admin Password
  Archives
  Set Date/Time
  Shutdown

Subpage: Statistics
Show the total number of votes submitted - include all categories.

Subpage: Site Access
This will allow the admin to enable/disable or change the PIN used to access the site.

Subpage: Category Control
Allow the admin to Start or Stop voting in one or all of the Categories. It should be clear from the buttons whether a category is currently enabled or not.

Subpage: Photo Settings
This will set the maximum number that can be used when voting. Default to "30"

Subpage: Competition Management
Assign a label for the current session. This label can be used to archive the current results (all categories). Clicking this should save all the current results in to the Archived_Results table, and then clear all the current votes.

Subpage: Admin Password
Set the password for the admin page.

Subpage: Archives
List all the archives currently in the table Archived_results. Provide a button to delete a particular archive - require confirmation before deleting.

Subpage: Set Date/Time
Use the Linux command "date -s *" to set date and time for the Raspberry Pi. Date will be in the format "dd-mm-yyyy". Time will be in the format "hh:mm".

Subpage: Shutdown
Clicking this will shutdown the Raspberry Pi - using the "halt" command. Require confirmation before shutting down.

Link: Logout
This link should log out the Admin account.

MySQL Tables:
  votes
        CREATE TABLE votes (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    category TEXT NOT NULL,
                    photo_number INTEGER NOT NULL,
                    timestamp DATETIME DEFAULT CURRENT_TIMESTAMP
                , vote_position INTEGER DEFAULT 1);

   settings
        CREATE TABLE settings (
                    key TEXT PRIMARY KEY,
                    value TEXT NOT NULL
                );

   archived_results
        CREATE TABLE archived_results (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    week_label TEXT NOT NULL,
                    category TEXT NOT NULL,
                    photo_number INTEGER NOT NULL,
                    vote_count INTEGER NOT NULL,
                    archived_date DATETIME DEFAULT CURRENT_TIMESTAMP
                , photographer_name TEXT, photo_title TEXT);
